# Data

This repository uses the Online Retail II dataset from the UCI Machine Learning Repository.

The dataset is publicly available at:

https://doi.org/10.24432/C5CG6D

After downloading the dataset, place the Excel file in this directory:

data/
└── online_retail_II.xlsx

The preprocessing script (`01_data_preprocessing.R`) loads the two transaction sheets and removes invalid purchase records before analysis.
